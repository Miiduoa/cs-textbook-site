Compression Basics

Compression reduces storage or transmission cost by encoding data more efficiently.

  • Lossless: original data fully recoverable (ZIP, PNG).
  • Lossy: discards some detail (JPEG, MP3).

Common Methods

MethodTypeIdea
HuffmanLosslessShort codes for frequent symbols
LZWLosslessDictionary of repeated patterns
JPEGLossyFrequency transform + quantization
MP3LossyPsychoacoustic modeling
