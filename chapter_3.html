Data Types

All data in a computer is stored as bits (0/1). A group of 8 bits is a byte. Different types of data require different encodings.

  • Integer: whole numbers (signed/unsigned).
  • Floating-point: real numbers with decimals (IEEE 754).
  • Character: letters/symbols (ASCII/Unicode).
  • Boolean: true/false.
