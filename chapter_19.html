English Reading Guide (中文導讀)

這章的英文重點是「流程」與「色彩」：pipeline, rasterization, RGB/CMYK。先記名詞，再看作用。

  • Rasterization 是「向像素轉換」。
  • Resolution = 寬 × 高。
  • RGB 是加法、CMYK 是減法。

Learning Objectives

  • Explain pixels, resolution, and file size trade‑offs.
  • Distinguish RGB, CMYK, and HSV color models.
  • Describe the rendering pipeline steps.
  • Convert RGB values to hex codes.

Key Terms

pixel, resolution, rasterization, RGB, CMYK, pipeline

Worked Example: RGB → Hex

RGB(255, 99, 71) = #FF6347.

Pixels & Resolution

Images are grids of pixels. Resolution = width × height. Higher resolution means more detail but larger files.

Color Models

RGB is additive (screens). CMYK is subtractive (printing).

ModelUse
RGBDisplays
CMYKPrint
HSVColor selection UI

Rendering Pipeline

Typical pipeline: Modeling → Transformation → Lighting → Rasterization → Display.

Interactive: RGB → HEX
