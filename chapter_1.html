Foundations of Computing

A computing system transforms input data into output results by following a finite, precise set of steps. In computer science, we separate the idea of a solution from its implementation.

  • Algorithm: a finite, unambiguous sequence of steps that solves a problem.
  • Program: an algorithm encoded in a programming language for execution.
  • Abstraction: hiding details to focus on essential behavior.
  • Model: a simplified formal system used to reason about computation.

These ideas allow us to prove what can be computed, estimate costs, and design systems that are correct and efficient.

Worked Example: From Problem → Algorithm

Problem: compute the sum of the first n positive integers.

  1. Initialize sum = 0
  2. For i = 1 to n: add i to sum
  3. Return sum

This algorithm runs in O(n) time. A mathematical abstraction gives n(n+1)/2, which is O(1). This contrast illustrates why abstraction matters.

The Turing Model

The Turing Model, proposed by Alan Turing in 1936, provides a theoretical definition of a computer. Before Turing, a "computer" was a person who calculated. Turing defined a machine that could perform any calculation that a human could, given enough time and paper.

A Turing Machine consists of:

  • A Tape: Infinite length, divided into cells, each containing a symbol.
  • A Head: Can read, write, and move left or right.
  • A State Register: Stores the current state of the machine.
  • A Table of Instructions: Tells the machine what to do based on the current state and symbol.

The Von Neumann Model

While the Turing model describes what can be computed, the Von Neumann model (1945) describes how to build a practical computer. It is based on the stored-program concept.

Computer Components

Modern computers are built from three main subsystems:

  1. Hardware: The physical equipment (chips, wires, screens).
  2. Software: The programs and data (OS, Apps).
  3. Data: The raw facts and figures managed by the software.

Social and Ethical Issues

As computers become ubiquitous, we face new challenges:

  • Privacy: Data collection vs. personal rights.
  • Security: Hacking, malware, and cyberwarfare.
  • Digital Divide: Gap between those with access to tech and those without.
