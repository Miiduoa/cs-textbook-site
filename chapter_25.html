Appendix C — Pseudocode

← Appendix B Appendix D →

Conventions

Pseudocode uses human‑readable constructs:

  • IF/ELSE, FOR, WHILE
  • Indentation to show blocks
  • Language‑agnostic operations

Example

Algorithm SumToN(n)
  total ← 0
  FOR i ← 1 TO n
    total ← total + i
  END FOR
  RETURN total

Interactive: Trace SumToN

Result: 55
