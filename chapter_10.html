Chapter 10 — Programming Languages

← Chapter 9 Chapter 11 →

Paradigms

  • Procedural: step-by-step (C, Pascal).
  • Object-Oriented: classes/objects (Java, C++).
  • Functional: pure functions, immutability (Haskell).
  • Logic: rules and inference (Prolog).

Compilation vs Interpretation

Compiled languages translate source code into machine code before execution. Interpreted languages execute line-by-line at runtime.

Type Systems

Types reduce errors by restricting operations. Systems may be static (compile-time) or dynamic (runtime).

Type SystemExampleNotes
StaticJava, CErrors caught earlier
DynamicPython, JSFlexible but runtime errors possible

Syntax & Semantics

Syntax is the grammar (form). Semantics is the meaning (behavior).
