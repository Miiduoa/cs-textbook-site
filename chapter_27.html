Boolean Algebra Laws

Boolean algebra uses binary variables (0/1). Key laws:

  • Identity: A + 0 = A, A · 1 = A
  • Complement: A + A' = 1, A · A' = 0
  • De Morgan's: (A + B)' = A' · B', (A · B)' = A' + B'
