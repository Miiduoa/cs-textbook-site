File Organization

  • Sequential: records stored in order; efficient for scans.
  • Indexed: index maps keys to record locations.
  • Hashed: hash function → bucket for fast lookups.

Blocks & Records

Files are stored as blocks (fixed size). A record may span multiple blocks, or multiple records may fit in one block.

Allocation Methods

MethodProsCons
ContiguousFast accessExternal fragmentation
LinkedEasy growthSlow random access
IndexedFast direct accessIndex overhead
