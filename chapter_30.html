Appendix H — Error Detection and Correction

← Appendix G Appendix I →

Parity Bit

A single bit added to make the total number of 1s even (even parity) or odd (odd parity). Detects single‑bit errors but cannot correct them.

Checksum

Sum all data bytes; the receiver recalculates and compares. Simple but limited error coverage.

Cyclic Redundancy Check (CRC)

Uses polynomial division. Common in Ethernet and storage. Detects burst errors effectively.

Hamming Code

Uses parity bits at power‑of‑2 positions. Can detect 2‑bit and correct 1‑bit errors.

For m data bits, need r parity bits: 2^r ≥ m + r + 1.
